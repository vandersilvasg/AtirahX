# 05_SCRIPT_LOVABLE_DASHBOARD_SHORT

```text
Crie um frontend SaaS para clinicas com estas telas:
- Dashboard, Agenda, Kanban CRM, Pacientes, Mensagens, Relatorios, Equipe, Financeiro, Acesso, Configuracoes.

Dashboard gestor deve ter:
- KPIs: comparecimento, no-show, confirmados/agendados, ocupacao, receita prevista 30d, tempo medio resposta lead.
- Graficos: comparecimento diario, no-show diario, ocupacao por medico, receita prevista vs realizada, funil por canal.
- Filtros globais: periodo, medico, unidade, canal, modalidade.

Diretrizes:
- layout premium e limpo;
- responsivo;
- componentes reutilizaveis;
- estados loading/empty/error;
- drawer contextual com acoes rapidas.

Entregue frontend navegavel completo e pronto para exportar.
```
